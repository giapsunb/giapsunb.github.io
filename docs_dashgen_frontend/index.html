Criando o ambiente de Desenvolvimento

Para executar a aplicação você precisa de:

  • Node.js
  • Yarn

Clone o repositório

git clone https://github.com/giapsunb/DashGen

Entre na pasta do projeto

cd DashGen

Altere para a branch da UI

git checkout dashgen-ui

Instalando as dependências do projeto

yarn install

Por fim, execute a aplicação

yarn start

Abra no seu browser http://localhost:3000 para ver a aplicação.

Boas práticas

Seguimos algumas práticas para manter um padrão no código do projeto .

1 - No seu editor instale a extensão do editorconfig. Com o editorconfig o seu editor vai automaticamente seguir o padrão de formato de arquivo do coding styles do projeto.

2 - Configure seu editor para usar eslint (Ferramenta de análise de código estática para identificar padrões e para aplicar um guia de estilo padrão).
